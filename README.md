# DataLightFlow

This repository is currently documented in Chinese.

Please see:

- [README-zh.md](./README-zh.md)
- [docs/技术实现方案.md](./docs/%E6%8A%80%E6%9C%AF%E5%AE%9E%E7%8E%B0%E6%96%B9%E6%A1%88.md)

DataLightFlow is a lightweight data pipeline for converting PDFs/images/PDF URLs to Markdown with local MinerU, then generating, evaluating, filtering, expanding, and exporting QA data for LLM training or RAG.
